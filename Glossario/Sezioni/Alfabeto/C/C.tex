\section{C}

\TG{Crawler}\DTG{ Detto anche “web crawler”, “spider” o “robot”, è un software che analizza i contenuti presenti nel web o in un database, in modo metodico ed automatizzato. In genere, si tratta di un bot che, una volta impostato ed in maniera del tutto autonoma, acquisisce dei dati utili per integrare o realizzare qualcosa (come un prodotto software).}
\TG{Crawling}\DTG{ Significa “fare scansione”, è il processo di raccolta delle informazioni e dei contenuti dal web o all’interno di un database e tale operazione viene effettuata da un bot (o crawler).}
